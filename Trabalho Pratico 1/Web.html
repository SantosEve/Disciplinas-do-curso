IFMG Sabará - Programação Web



Programação Web é uma disciplina da Proposta Pedagógica Curricular (PPC) do Curso Técnico de Informática Integrado do IFMG Campus Sabará, ministrada atualmente pelo professor Carlos Alberto Severiano Junior, essa disciplina tem como ementa:

Unidade 1:

  • Como funciona a Web.
  • Estruturando páginas web com HTML5: textos, listas, imagens, tabelas, formulários e links.

Unidade 2:

  • Estilizando páginas com a linguagem CSS3.
  • Cores e posicionamento.
  • Técnicas de layout.

Unidade 3:

  • Dinamizando páginas web com a linguagem Javascript.
  • Manipulando elementos por meio do DOM (DocumentObjectModel).
  • Tópicos: design responsivo, acessibilidade na web e padrões de desenvolvimento web.

Introdução ao HTML:


HTML é a abreviação para HyperTextMarkupLanguage (Linguagem de Marcação de Hipertexto). Esta é a linguagem usada para criar os websites que visitamos diariamente, inclusive este aqui. Ele provê uma forma lógica de estruturar o conteúdo das páginas Web.
Sua Sintaxe contém as tags:
  • "doctype html" - Para criar o documento html.
  • "html lang=" - Para declarar a linguagem utilizada, por exemplo: 'html lang="pt-br"' declarando o uso da Língua Portuguesa do Brasil.
  • "head" - Para criar a "cabeça" do site, ou seja, o que vem nas primeiras linhas.
  • "body" - Para criar o "corpo" do site, ou seja, o que virá após a "cabeça".
  • "html" - Para declarar o fim das linhas do código e mostrar que o documento acaba ali.
    • É importante destacar que tags devem ser abertas com "< >" e fechadas com "< / >"


    HTML é a linguagem de marcação que torna possível determinar onde cada item do site deve estar posicionado, mas se deseja estilizar sua página web é necessário utilizar o CSS3, Cascading Style Sheets (CSS) ou Folhas de Estilo em Cascata, é um simples mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento Web. Portanto, seu objetivo é definir o visual de uma página Web.

    Já o Javascript, é uma linguagem de programação, ou seja, envolve lógica de programação. Porém ainda não foi abordado isso em sala de aula porque estamos na Unidade 2.
